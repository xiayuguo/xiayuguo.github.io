Markdown中实现合并单元格

Dec 10, 2018

Method

  • 使用 <br> 换行符(隐性换行)
  • 使用 html 标签 rowspan 和 colspan excel 转 html

Raw

<table>
    <tr>
        <th rowspan="2">值班人员</th>
        <th colspan="2">星期一</th>
        <th>星期二</th>
    </tr>
    <tr>
        <td>李强</td>
        <td>张明</td>
        <td>王平</td>
    </tr>
</table>
