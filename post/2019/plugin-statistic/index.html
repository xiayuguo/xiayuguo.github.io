Plugin Statistic

Jan 6, 2019

统计项目中的有效代码行数和注释是一件很有趣的事, 安利大家一款统计代码行数的插件 statistic

安装

  1. 进入 File -> Settings -> Plugins
  2. 搜索关键字 statistic -> Search in repositories
  3. 选中插件 -> Install
  4. 安装成功后重启 Pycharm

使用

以 Flask 为例

  1. 底边栏 statistic

  2. Refresh

  3. Detail

表头 描述
Source File 单个文件名
Total Lines 单个文件总行数
Source code Lines 纯代码行数
Source code Lines[%] 纯代码行数百分比
Comment Lines 注释行数
Comment Lines[%] 注释行数百分比
Blank Lines 空行行数
Blank Lines[%] 空行行数百分比
