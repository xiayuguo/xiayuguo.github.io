Vim Tutorial

Jan 28, 2019

从 2015 年使用 Vim 至今已经有 4 个年头了，每次都是碰到一个不会的操作现查，加上不是特别常用，于是促成了“半吊子”的诞生(/ω＼)，近几日偶然间发现 Vim 自带教程，贴出来和大家分享下。

  1. 进入终端 Terminal 执行命令 vimtutor
洋洋洒洒 30 分钟的简易教程，大家可以手动见证
  1. 关于上述内置教程的位置，大家可以通过 vimtutor --help 查看(请注意只有几秒，一闪而逝), 我的位置是 /usr/share/vim/vim74/tutor
