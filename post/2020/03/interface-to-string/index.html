Golang 中将接口转换为字符串（译文）

Mar 15, 2020

作者：Stefan Nilsson

原文网址：https://yourbasic.org/golang/interface-to-string/

使用 fmt.Sprintf 将interface value转换为字符串。

var x interface{} = "abc"
str := fmt.Sprintf("%v", x)

实际上，可以使用相同的技术来获取任何数据结构的字符串表示形式。

var x interface{} = []int{1, 2, 3}
str := fmt.Sprintf("%v", x)
fmt.Println(str) // "[1 2 3]"
