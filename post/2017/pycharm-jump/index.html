Pycharm实现前进/后退

Apr 19, 2017

操作系统: Ubuntu14.04; Pycharm版本: 2016.3; 这里的前进指的是追踪函数源码, 后退是前进的反方向。

前进

  • Ctrl+B
  • Ctrl+Q+<F4>

后退

  • View -> ToolBar -> √ ToolBar(出现工具栏，工具栏中有前进和后退)
  • Ctrl+ Alt + Left(我试了没生效，不知道为啥)
  • 罗技G502鼠标左侧前进键和后退键完美兼容Pycharm(意外之喜)
